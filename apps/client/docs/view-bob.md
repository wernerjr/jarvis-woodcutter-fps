# View bob / sway (client-only)

- O jogo aplica bobbing/sway **apenas nos modelos na mão** (tool pivot), para dar sensação de FPS.
- A câmera continua sendo a referência authoritative para raycasts/interação/dano.

## Toggle
- No menu de pausa: botão `View bob: ON/OFF`
- Preferência é salva em `localStorage` (`woodcutter_view_bob`).

Jarvis — Robô Lenhador

FPS simples: corte árvores, elas reaparecem depois de um tempo.

Dica: clique para capturar o mouse (Pointer Lock). ESC abre o menu de pausa.

Pausado

ESC para retomar / abrir pausa.

Controles

  • WASD: mover
  • Mouse: olhar
  • Clique esquerdo: cortar (perto da árvore)
  • ESC: pausar / liberar mouse

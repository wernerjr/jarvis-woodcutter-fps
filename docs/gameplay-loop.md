# Gameplay loop (visão curta)

1) **Cortar árvores** (machado) → loot direto no inventário (tronco/galho/folha).
2) **Coletar pedras** (mão) → usado em crafting.
3) **Crafting básico** (tecla C) → tocha, ferramentas de pedra, fogueira, fornalha (forja).
4) **Mina** → entrar via portal, minerar nós de ferro com picareta → minério de ferro.
5) **Fornalha (Forja)** → colocar combustível + minério e iniciar fundição → barras de ferro.
6) **Mesa de Forja** → usar barras para forjar ferramentas de metal.

Notas:
- Inventário: 20 slots, stacks até 100, overflow é descartado com toast.
- Simulação congela fora do modo playing, com exceções explícitas (forja aberta).
